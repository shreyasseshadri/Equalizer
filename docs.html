User Documentation


Introduction

  • This web application enables the user (student) to keep track of the amount spent for mess food and hence, the refund they need to receive by the end of the semester, along with daily menu of the chosen mess. It offers auto-scaling and high availability, supports all platforms as it works on browser. Learn how to use the application with this documentation.

    • Getting Started
    Sign-Up & Log-in

  • A valid email address and password is required for signing up for the application. After entering the required details in the 'Sign-Up' form, which can be accessed through the top navigation bar's 'SignUp' link, a verfifcation link is sent and you are required to press a button to verify and hence, activate your account.
  • You would then be redirected to the login page which can also be accessed through the top navigation bar. After filling in the details and clicking on the button, you would be logged in and redirected to edit your profile details.
  • In case of forgotten password, you can set a new one by clicking on the link available on the page saying 'forgot password', and continuing from the login page after you're done.
    • After Log-in

  • The profile details page allows you to edit your details as you see fit. It re-directs you to the home page after updating the details. You can always come back to this page navigating from the sidebar, which can be seen at the top left side of the navigation bar.
  • The home page provides you with the basic details of the application along with the respective contacts in case of any issues. We assure you that the response would be immediate.
  • There are four functions that the side navigation bar does for you.

    • Navigate to edit your profile details through 'Profile Details' link.
    • Navigate to your user dashboard which is filled with mess and refund details.
    • Finally, you can deactivate your account if required by navigating through the 'Account Status' link. This means that you are opting out of the mess.
  • The functions the top navigation bar does for you are:

    • Logging in/out when required - right side.
    • Sign-up for new users - right side.
    • Username which redirects you to profile details page - right side.
    • The application logo redirecting you to the home page - left side.
    • Three menu bars icon toggling the side navigation bar - left side.
    • Handling User Dashboard

  • This page contains a timeline that indicates when the meals are supposed to be. For example: Breakfast, lunch, and so on, as you will see on the page.
  • The timeline has 'dots' that when clicked, show you the menu for the respective mealtime, giving you an idea of what's being served in the mess currently.
  • It also has two cards-like structures, which show how much of the refund you have to receive based on your expediture with respect to meals had in mess.
  • The last element is the button 'Scan QR' which does basically what the side navigation bar's 'Scan QR' does.

    • Scanning The QR Code

  • You can access this page from either the button in user dashboard. After pressing the link/button, the application would need permission to access your camera to scan the QR code of whichever mess you want. Currently, there are a total of ten messes in the database, eight boys' and two girls'. Each of them have a unique QR code and you can scan any one of them depending on your preferences. The cost of the meal would be updated on the cards in the dashboard showing you the required details.
